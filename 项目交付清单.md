## 📝 使用说明

### 启动程序
```bash
# 管理员权限运行
python src/main.py
```

### 基本操作

#### 1. 查看路由
- 启动后自动显示系统路由
- 使用网关筛选查看特定网关的路由
- 默认显示192.168.1.1的18条路由

#### 2. 添加路由
```
1. 点击绿色"新增路由"按钮
2. 输入目标IP (例如: 8.8.8.8)
3. 确认网关 (默认192.168.1.1)
4. 点击"添加并应用"
```

#### 3. 删除路由
```
1. 找到要删除的路由 (NetMgmt协议)
2. 点击红色"删除"按钮
3. 确认删除操作
```

#### 4. 筛选路由
```
使用网关筛选下拉菜单:
- All (全部) - 44 条
- 192.168.1.1 - 18 条 (默认)
- 其他网关...
```

## ⚠️ 注意事项

### 使用限制
1. ✅ 需要管理员权限运行
2. ✅ 仅支持 Windows 10/11
3. ✅ 仅支持 IPv4 路由
4. ✅ 操作会影响系统网络

### 安全提示
1. ⚠️ 删除路由前请确认
2. ⚠️ 建议先在测试环境使用
3. ⚠️ 重要路由请做好备份
4. ✅ 程序会自动创建快照

### 故障排除
1. **启动失败**: 检查管理员权限
2. **路由不显示**: 点击"刷新"按钮
3. **添加失败**: 检查网关和接口
4. **删除失败**: 只能删除静态路由(NetMgmt)

## 🆘 支持与反馈

### 问题反馈
- 查看日志: `logs/` 目录
- 检查配置: `profiles/home.json`
- 查看文档: `docs/USER_GUIDE.md`

### 常见问题
1. Q: 为什么需要管理员权限?
   A: 修改系统路由表需要管理员权限

2. Q: 可以删除系统路由吗?
   A: 只能删除静态路由,系统路由被保护

3. Q: 路由会保存吗?
   A: 新增的路由默认是永久路由(persistent)

4. Q: 如何备份路由?
   A: 使用导出功能或Profile功能
